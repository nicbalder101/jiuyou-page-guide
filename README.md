# jiuyou-page-guide

## 项目简介

`jiuyou-page-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面资源的集合，方便统一管理与分发。

仓库中的页面内容均以 HTML 格式存储，可直接在浏览器中打开查看，或通过 GitHub Pages 等静态托管服务进行部署。

## 目录说明

```
jiuyou-page-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个独立 HTML 页面建议放置在此目录下，可按功能或类型进一步划分子目录。
- `assets/`：存放页面间共享的样式、脚本或媒体资源，避免重复引用。

## 页面归档说明

本仓库中的页面均经过人工整理与归档，确保：
- 页面结构完整，可直接运行。
- 不包含外部不可控资源的硬编码链接。
- 不针对任何具体网站或服务，保持通用性。

如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保所有资源引用路径正确（推荐使用相对路径或仓库内资源链接）。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面均会记录在 Git 提交历史中。
- 欢迎通过 Issues 提交问题或建议，但请注意，本仓库不接受任何形式的商业推广或敏感内容。
- 如需 fork 或二次分发，请遵守仓库所使用的开源许可证（见 LICENSE 文件）。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击 `pages/` 下的 HTML 文件，即可在浏览器中查看。
3. 若需部署到 Web 服务器，请将整个仓库内容上传至服务器根目录，并确保 `index.html` 作为默认首页（如有）。
